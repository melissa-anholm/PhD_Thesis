	{\todo[caption={#2}, size=\scriptsize, color=orange, #1]{\renewcommand{\baselinestretch}{1.0}\selectfont#2\par}}  % default spacing seems to be like 1.2.  Change to 1.0.  This also allows you to override the color selection.
